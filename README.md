## :sparkler: Overview

時間に応じてレベルアップする学習報告SPA

## :earth_americas: AppURL

https://polpa.work

## :pencil2: Description

- 学習時間記録
- 報告された時間に応じてレベルアップ
- 学習記録へのコメントやいいね
- タグ付けによる管理
- 他のユーザーをフォロー
- ゲストログイン
- タグ・ユーザー・学習記録を検索
- ユーザーへの通知
- 通知設定
- ユーザーランキング
- 週間目標
- タグをフォロー
